>玩法 : 一分钟内射击的乌鸦数量得到的得分，射击的越多，乌鸦移动的速度越快，得分越困难

这个游戏叫eliminate crow 主UI界面和所有图片都是网上找的素材之后，自己用PS合成的最终图片，然后制作成风格相似的一套图片（3张），主界面有两个按钮（按钮是网上素材直接找的），一个是start点击开始游戏，一个是游戏说明按钮。游戏过程中，也会有许多各种各样的声音。


### 制做这款简易的射击游戏的时候，大概经历四次修改， 
第一次是制作的过程，最困难就属于点击事件了，老是有误差，还有就是图片大小啊，图片不适应啊等等，第一次是只有游戏界面，没有主页面和说明界面

第二次游戏做出来了，并且有了游戏说明和主页面，但是发现60秒时间走完了，但是乌鸦还是没有消失，后来在remove舞台上不需要的东西和关掉事件函数。

第三次完成后，发现不同的电脑分辨率，随着滚动条移动，鼠标和瞄准镜重合不了，所以后来加上了关于scrillTop兼容不同分辨率的电脑
 
第四次修改主要就是删减代码，显得没那么臃肿。

### 主界面

主界面是有一张背景图和两个按钮，所有的图片都是事先用loadManifest加载进去的，然后complete完成后调用函数。按钮可以分别点击进入别的界面，本游戏全部使用createJS制作。按钮界面是事件点击，然后分别用tween类来实现画面之间的跳转，跳转的方式bounceOut，backOut等等，使画面没那么生硬，看起来生动一点。
